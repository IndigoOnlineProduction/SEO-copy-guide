#SEO Copy Guide

##TAGS
```
?ip=NAME OF SHOP&tile=seocopy_NAME OF LINK
```

###I.E:
```
* ?ip=holidayguide&tile=seocopy_christmascards

* ?ip=kidsgiftguide&tile=seocopy_herogifts
```

###E.G.
``` 
ORIGINAL LINK: /holiday/blogger-shop/sasha/

SEO LINK: https://www.chapters.indigo.ca/en-ca/holiday/blogger-shop/sasha?ip=bloggershop&tile=seocopy_sashaexeter
```
